# Renewlet v0.2.2

Renewlet v0.2.2 adds account security features: protect password sign-in with authenticator codes, keep one-time recovery codes as a backup, and sign in directly with passkeys.

[中文 ->](./v0.2.2-zh.md)

---

## Read Before Upgrading

Cloudflare one-click deploy users should still upgrade from the generated repository by running `Sync Renewlet Upstream`. Do not click the one-click deploy button again. If an older generated repository does not have this entry yet, add it once from the [Cloudflare deploy documentation](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.md#existing-one-click-deploy-users).

---

## New Features

- Turn on an authenticator in Settings to require a 6-digit code at password sign-in and generate one-time recovery codes as a backup.
- Add passkeys to sign in directly with your browser, device biometrics, or a security key.
- Admins can reset a user's authenticator or passkeys separately from the user management page.
