# Renewlet v0.1.9

Renewlet v0.1.9 继续打磨家庭共享费用分摊（[zhiyingzzhou/renewlet#34](https://github.com/zhiyingzzhou/renewlet/pull/34)，感谢 [@Shy0o](https://github.com/Shy0o)）。自定义分摊现在更贴近实际收款场景：你只需要填写每位成员应承担的金额，Renewlet 会统一显示成员合计、你的份额和可回收金额。

[English ->](./v0.1.9-en.md)

---

## 升级前请先读

Docker 用户从 v0.1.8 升级不需要手动迁移。

Cloudflare 用户请按现有部署流程发布；本版本包含家庭共享费用分摊所需的数据迁移。

Cloudflare 一键部署用户仍然从生成仓库运行 `Sync Renewlet Upstream` 升级，不要重新点一键部署按钮。旧生成仓库如果还没有这个入口，按 [Cloudflare 部署文档](https://github.com/zhiyingzzhou/renewlet/blob/main/docs/cloudflare-workers-deploy.zh-CN.md#旧一键部署用户) 一次性补上。

---

## 优化改进

- 自定义分摊不再要求成员金额凑齐订阅价格；适合按实际应收金额记录家庭、朋友或团队共享订阅。
- 费用摘要更清楚地区分成员合计、你的份额和可回收金额。
- 管理成员从订阅表单里独立出来，成员较多时也不会把表单撑得很长。
- 成员使用不同货币时，订阅表单、卡片、详情、统计和 CSV 导出的金额口径保持一致。
- Cloudflare 部署文档进一步明确一键部署和手动部署的升级路径。

---

## Bug 修复

- 修复自定义分摊金额与订阅价格不一致时，可回收金额显示不准确的问题。
- 修复管理成员时订阅弹窗可能被意外关闭或焦点异常的问题。
